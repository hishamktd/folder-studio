# Quick Start Guide

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)

## Production

```bash
# Build for production
pnpm build

# Start production server
pnpm start
```

## How to Use

1. **Upload an Image**
   - Click "Upload Image" button
   - Or click "Paste from Clipboard" if you have an image copied

2. **Customize Your Folder**
   - Enter a title in the "Folder Title" field
   - Select a font weight (Light to Extra Bold)
   - Choose colors for the title background and text using color pickers

3. **Choose a Folder Style**
   - Select from 8 preset styles (Modern Red is default)
   - Each style has unique colors and effects

4. **Export Your Icon**
   - Click any export size button (Small, Medium, Large, Extra Large)
   - Your custom folder icon will download as a PNG file

## Tips

- The preview updates in real-time as you make changes
- Use paste from clipboard for quick workflow
- Try different folder styles to match your aesthetic
- Export multiple sizes if needed for different use cases
- The Modern Red style matches the reference design

## Folder Styles

| Style | Description |
|-------|-------------|
| Modern Red | Professional red folder with shadow (reference style) |
| Neon Blue | Cyan folder with neon glow effect |
| Neon Purple | Purple folder with neon glow effect |
| Neon Pink | Pink folder with neon glow effect |
| Glass | Translucent glassmorphism effect |
| Matte Dark | Dark professional matte finish |
| Gradient Sunset | Orange to yellow warm gradient |
| Gradient Ocean | Teal to cyan cool gradient |

## Export Sizes

- **Small (128x128)**: Compact displays, mobile
- **Medium (256x256)**: Standard folder icon size
- **Large (512x512)**: High-resolution displays
- **Extra Large (1024x1024)**: Retina/4K displays, maximum quality
